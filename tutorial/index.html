Tutorial

Basic settings

The pipeline is designed to run with a minimum of three supplied parameters:

* path to FASTQ sequence data (-f)
* name and path of multi-FASTA target file (-t)
* path to folder in which outputs should be placed

FASTQ sequence data

The FASTQ data can be provided in one of two ways:

1) Raw FASTQ output from the MinION

  • supply the path of the folder containing the FASTQ files

2) Trimmed FASTQ file

  • supply the name and path of the FASTQ file

Target file

The multi-FASTA target file must have the name of the target as the header e.g.:

>NOS3
CCCGA......

>e35S
GGTCC.....

Additionally, for best results you should provide a second file in the same folder as the target file. This file must be named 'genesofinterest_wheat.fasta'. It must have the same FASTA sequences as the original target file, as well as any wheat sequences you wish to find. Including the wheat sequences in the original target file causes a very large number of reads with only wheat sequence to be baited and analysed. This is not desired.

If you do not provide this file, the pipeline will still work, however, it will be harder to find any junctions between wheat and the cassette.

The following command will run the pipeline on the supplied sequences with default parameters

minion_amplicon_pipeline.py /path/to/outputfolder -f /path/to/sequences -t /path/to/targets/targets.fasta

Optional parameters

There are a number of optional parameters than can be supplied to the minion_amplicon_pipeline.py script

usage: minion_amplicon_pipeline.py [-h] [-l LENGTH] -t TARGETFILE
                                   [-k KMERSIZE] [--hdist HDIST]
                                   [-r READLENGTH] [-f FASTQ] [-o OVERHANG]
                                   [-a]
                                   path

Pipeline for transgene analyses

positional arguments:
  path                  Specify path

optional arguments:
  -h, --help            show this help message and exit
  -l, --length LENGTH
                        Minimum length of match. Default is 50
  -t, --targetfile TARGETFILE
                        Name and path of multi-FASTA file of targets
  -k, --kmersize KMERSIZE
                        Size of kmer to use when baiting reads against targets
  --hdist HDIST         Number of mismatches to allow for baiting. Default is 2.
  -r, --readlength READLENGTH
                        Minimum read length for canu
  -f, --fastq FASTQ
                        Supply either the name and path of FASTQ file to process, or the path 
                        of raw FASTQ files to combine and trim prior to processing
  -o, --overhang OVERHANG
                        The minimum length a 5' or 3' overhang must be to be considered
  -a, --assemble        
                        Optionally assemble binned reads
